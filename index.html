Library Organizer

GitHub Repository

Details

Language: C++
Specs:
  • Read Input from text file
  • Prompt the user for a menu selection
  • Display an organized list from the input file
  • Validate, then add a new list item based on user-input, update index
  • Validate, then remove a list item based on user-input, update index
  • Validate, then search the list based on user-input
  • Save all entries/removals back to the text file and exit the program


An application for organizing libraries of various
types (ie. music, book, photos, contacts), demonstrating
linked lists in C++. Upon running the application,
the program reads input from a text file, then prompts
the user with a menu of operations they can perform on
the library.
